### DynamoDBのデータ整合性モデルの正しい説明を選択してください

正解：Scanなど（Read）のクエリを実行する際などにオプション機能で強い整合性モデルを利用することができます。

### 次のうちAmazon Auroraについて正しい内容を選択してください

正解：マルチAZに跨って仮想ボリュームを構成されています。

### オンプレミス環境からAWSへとデータ移行が必要です。Amazon EFS ファイルにデータ移行するために利用するべきサービスはどれでしょうか？

正解：AWS DataSync

### グローバルに展開されたアプリケーションに対して高速の読み取り/書き込み処理を提供するDynamoDBのレプリケーションの仕組みを選択してください

正解：グローバルテーブル

### API Gatewayの特徴として正しい説明を選択してください

正解：APIを生成・管理することができる, WEBSocketを利用した双方通信が可能である, EC2／Lambda／任意のウェブアプリケーションのワークロード処理を実行する,
最大数十万個のAPI同時呼び出し・受付が可能, AWS Identity and Access Management (IAM) とAmazon Cognito を使用して API へのアクセスを許可,
Amazon CloudFront との連携, Amazon CloudWatch を使用してサービスへの呼び出しを視覚的にモニタリング

### あなたはコールセンターシステムを開発しており、音声をテキストに変換する文字起こし作業が必要です。どのサービスを利用するべきでしょうか？

正解：Amazon Transcribe

### Amazonのコンテナサービスの説明として正しい内容を選択してください

正解：Amazon Elastic Container Registry (ECR) は、完全マネージド型の Docker コンテナレジストリです。（保存、管理、デプロイ）

### AWSのコスト試算について正しい内容を選択してください

正解：AWS 料金計算ツール（AWS Pricing Calculator）はすべてのAWSリソースに対するニーズに応じたコスト試算ができます。

### 次のうちCloudWatchの特徴として正しい内容を選択してください（２つ選択してください。）

正解：機械学習により異常値（アノマリー）を自動検出できます。, CloudWatch RUMを利用することでWEBアプリケーションのクライアント目線でのパフォーマンス問題の解析や解決に約立つモニタリングを実施することができます。

### 次のAWSサービスのうちで、リソース停止後に自動的に再起動するサービスを選択してください。（2つ選択してください。）

正解：Elastic Beanstalk, AWS OpsWorks

### あなたはソリューションアーキテクトとしてデータベースを選定しています。ElastiCacheを使用するユースケースはどのような状況ですか？ （2つ選択してください。）

正解：インメモリデータベースを利用したい。, セッションデータ等の高速解析処理

### AWSサービスを利用する際に、AWSから禁止された利用方法を確認する方法はどれですか？

正解：AWS Acceptable Use Policy（利用規約）

### ある金融機関のRDSデータベースにクレジットカード情報や個人情報に絡んだデータを大量に保存しており、これらを解析してデータ漏洩などがないように保護することが必要です。利用できるサービスはどれでしょうか？

正解：Amazon Macie

### 次の中でIoTデータの蓄積用のデータベースとして最適なサービスはどれでしょうか？（2つ選択してください。）

正解：Amazon Timestream, Amazon DynamoDB

### AWSのセキュリティに係る責任共有モデルによると、顧客が実施すべきである「継承される統制」に含まれるのはどれでしょうか？ （2つ選択してください。）

正解：物理的統制, 環境統制

### あなたはモバイル対応の動画アプリケーションをAWS上で構築しています。S3に保存されているデータをモバイルデバイスでの動画再生を実行するために変換処理が必要です。どのサービスを利用するべきでしょうか？

正解：AWS Elemental MediaConvert

### ユーザーが過去の請求情報を確認するために利用できるAWSサービスはどれでしょうか？

正解：AWS Billing and Cost Management
